# Обнаружение девиантного поведения на видео (YOLOv11 + DBSCAN + UniFormer)

Трёхэтапный пайплайн для детекции драк на видеопотоке:

1. **YOLOv11** — детекция и трекинг людей в кадре.
2. **DBSCAN** — пространственная кластеризация боксов с адаптивным `eps`
   (среднее значение диагонали бокса), чтобы объединить близко
   расположенные/перекрывающиеся объекты в один регион перед классификацией.
3. **UniFormer** — пространственно-временная классификация по клипам
   из 16 кадров (5 FPS), вырезанным по объединённому боксу.

Решение по треку усредняется по `CLIPS_PER_DECISION` последовательным
предсказаниям клипов, что снижает количество ложных переключений метки
на отдельных шумных кадрах.

## Структура репозитория

```
violence-detection/
├── README.md
├── requirements.txt
├── .gitignore
└── src/
    └── fight_recognition.py   # основной скрипт инференса
```

Конфиг классификатора, чекпоинт и файл меток в репозиторий не включены
(слишком большие для git) — они передаются путями через CLI-аргументы.

## Установка

```bash
python -m venv .venv
source .venv/bin/activate          # Windows: .venv\Scripts\activate

pip install -r requirements.txt

# mmaction2 ставится отдельно через инструмент OpenMMLab (mim),
# обычные requirements.txt/pip install с ним не справляются стабильно
pip install -U openmim
mim install mmaction2
```

Требуется PyTorch с поддержкой CUDA, если планируется запуск на GPU —
ставьте версию `torch`/`torchvision` под свою версию CUDA отдельно,
см. https://pytorch.org/get-started/locally/

## Использование

```bash
python src/fight_recognition.py \
    CONFIG_PATH \
    CHECKPOINT_PATH \
    LABEL_PATH \
    VIDEO_PATH \
    --device cuda:0 \
    --yolo-version yolo11m.pt \
    --output output/result.mp4
```

| Аргумент | Описание |
|---|---|
| `config` | путь к test-конфигу mmaction2 для UniFormer |
| `checkpoint` | путь/URL к чекпоинту классификатора |
| `label` | путь к текстовому файлу с метками классов |
| `video` | путь к входному видео |
| `--device` | `cuda:0` или `cpu` |
| `--yolo-version` | веса YOLO для детекции людей |
| `--output` | путь к аннотированному видео на выходе |

## Результаты

Финальная конфигурация (YOLOv11 + DBSCAN + UniFormer на кропах) на тестовой
выборке из 133 эффективных клипов: ROC-AUC ≈ 0.999, AP ≈ 0.999
(TP=66, FP=2, FN=1, TN=64). Подробное описание экспериментов, сравнение
архитектур и обоснование выбора — в тексте ВКР, главы 2–3.
